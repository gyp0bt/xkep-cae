# mesh/ — メッシュ生成モジュール

[← README](../../README.md) | [← roadmap](../../docs/roadmap.md)

## 概要

撚線構造のメッシュ生成を担当する。
ヘリカル配置の素線メッシュ、被膜モデル、シースモデルを生成。

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `twisted_wire.py` | `TwistedWireMesh` — ヘリカル撚線メッシュファクトリ（3/7/19/37/61/91/1000本対応） |
| `ring_compliance.py` | リングコンプライアンス行列（Fourier閉形式、シース剛性表現） |
| `tube_mesh.py` | チューブメッシュ生成（HEX8要素向け） |

## 撚線メッシュ

`make_twisted_wire_mesh()` で指定本数の撚線メッシュを生成。

### 対応構成

| 構成 | 素線配置 | 接触ペア |
|------|---------|---------|
| 3本 | 1+2（中心+外周） | 3 |
| 7本 | 1+6 | 21 |
| 19本 | 1+6+12（3層） | 112 |
| 37本 | 1+6+12+18（4層） | — |
| 61本 | 1+6+12+18+24（5層） | — |
| 91本 | 1+6+12+18+24+30（6層） | 66,066 |
| 1000本 | 多層 | ~7,335,879 |

### 付属モデル

- **CoatingModel**: エナメル被膜（接触半径拡大、弾性ばね）
- **SheathModel**: PE/PVCシース（解析的リングコンプライアンス、Fourier近似内面）

## 計測済みスケーリング

| 素線数 | 要素数 | DOF | メッシュ生成時間 |
|---:|---:|---:|---:|
| 91 | 364 | 2,730 | 0.001s |
| 1000 | 4,000 | 30,000 | 0.006s |

> メッシュ生成はボトルネックにならない。broadphase・接触NR収束が律速。
