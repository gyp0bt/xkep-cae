## 概要

シース（外被）モデルの挙動設計を確定。前回 status-056 で実装した幾何・断面特性の計算基盤に対して、シースの力学的挙動（素線食い込み・変形・有限滑り）をどのようにモデル化するかの設計方針を策定。

## 課題と設計判断

### 課題

シースへの素線食い込み（強接触）を表現するには、シースの局所変形を解く必要がある。

| アプローチ | 問題点 |
|-----------|--------|
| 連続体要素でシースを離散化 | 計算コスト大（薄肉シースに多数要素） |
| シェル/梁要素で離散化 | せん断自由度なしだとロッキング |
| 凹包内面の直接モデル | 接触不連続性による数値不安定 |

### 採用設計: 解析的リングコンプライアンス行列 + ペナルティ接触

シースをFE離散化せず、解析的リング理論で剛性を表現する。

#### 原理

シース断面は z=const 切断面で「厚肉弾性リング」として扱う。N 本の最外層素線からの径方向集中荷重を受けるリングの変形を Fourier 級数で解析的に解く。

```
励起モード（7本撚り/外層6本）: n = 0, 6, 12, ...
励起モード（19本撚り/外層12本）: n = 0, 12, 24, ...

モード 0（膜変形）: c₀ = f(E, ν, a, b)    — 均等圧縮
モード n≥2（曲げ）:  cₙ = f(E, ν, a, b, n) — 素線間の局所凹み

コンプライアンス行列: δr = C · F_contact   (N×N 行列)
```

#### 設計の利点

| 利点 | 説明 |
|------|------|
| **DOF追加ゼロ** | シースは解析解で表現、FE離散化不要 |
| **ロッキングなし** | 要素がないのでロッキングの概念自体が不在 |
| **せん断含む** | 厚肉リング理論で自動的にせん断変形を含む |
| **接触点間の連結** | C の非対角項で周方向カップリング表現 |
| **膜厚分布対応** | 各 Fourier 係数に t(θ) の影響を含められる |

#### シース内面: Fourier 近似

素線配置から内面形状を直接計算するのではなく、Fourier 近似で滑らかに表現:

```
r_inner(θ) = R₀ + Σ aₙ cos(nθ) + bₙ sin(nθ)
```

7本撚りなら a₆ が支配的。多角形凹包の不連続は Fourier 打切りで自然に平滑化される。

#### シース外面: 円形

外面は円形で、膜厚分布 `t(θ) = r_outer - r_inner(θ)` が周方向に変化する。

#### 有限滑り

- **シース-素線/被膜**: 素線の接触位置 θ_contact は変形とともに移動。C 行列の接触点列を再配置。摩擦は既存の `friction_return_mapping` を流用（径方向=法線、周方向+軸方向=接線）。
- **シース-シース**: 外側シース外面（円形）同士の接触。円-円の接触として既存の梁-梁接触フレームワーク（`ContactPair`）をそのまま流用可能。

## 段階的実装計画（Stage S1〜S4）

| Stage | 内容 | 検証方法 |
|-------|------|----------|
| **S1** | 均一厚リングの解析的コンプライアンス行列（モード 0 + n の閉形式） | FEMリング解との比較 |
| **S2** | 膜厚分布 t(θ) の Fourier 導入 + 修正コンプライアンス | Fourier 係数の収束性検証 |
| **S3** | 有限滑り（θ 再配置 + 摩擦、既存 friction_return_mapping 流用） | 撚線引張でのシース-素線滑り |
| **S4** | シース-シース接触（円-円ペナルティ、既存 ContactPair 流用） | 撚撚線のサブケーブル間接触 |
