## 1. 推奨ソルバー構成（リファレンス構成）

### 背景

S3ベンチマーク（status-089〜097）はAL法ソルバー（`solver_hooks.py`）で実施されたが、
NCPソルバー（`solver_ncp.py`）は以下の構造的優位性を持つ:

- **Outer loop 不要**: 変位 u とラグランジュ乗数 λ を同時更新 → AL乗数の暴走問題が構造的に不在
- **Alart-Curnier 鞍点定式化**: 活性セット判定が NCP 関数で自動化、k_pen は正則化のみ
- **摩擦対応済み**: Coulomb return mapping + NCP ハイブリッド
- **Line contact + Mortar 統合済み**: Gauss 積分力評価 + Mortar 離散化

AL法で必要だった `adaptive_omega`, `lambda_n_max_factor`, `best-state fallback` 等の
ワークアラウンドがNCPソルバーでは不要。

### リファレンス構成（今後のベンチマークはこの構成を基準とする）

| 項目 | 設定 |
|------|------|
| **ソルバー** | `newton_raphson_contact_ncp`（`solver_ncp.py`） |
| **接触離散化** | Line-to-line Gauss 積分（`line_contact=True`） |
| **NCP関数** | Fischer-Burmeister（`ncp_type="fb"`） |
| **摩擦** | Coulomb return mapping + NCP ハイブリッド（`use_friction=True`） |
| **Mortar** | 適応ペナルティ付き Mortar 離散化（`use_mortar=True`） |
| **同層除外** | `exclude_same_layer=True`（層間接触のみ） |
| **k_pen** | 自動推定（beam EI ベース） |
| **線形ソルバー** | DOF閾値に応じて自動切替（直接法 / GMRES+ILU） |
| **前処理** | ブロック前処理 Schur 補集合（大規模問題時） |

### 比較用サブ構成（リファレンスとの差分検証用）

必要に応じて以下のサブ構成との比較を行う:

| 構成名 | 差分 | 用途 |
|--------|------|------|
| **AL-Uzawa** | `solver_hooks.py` + adaptive omega | レガシー比較、収束性検証 |
| **NCP-PtP** | `line_contact=False` | PtP vs Line contact 精度比較 |
| **NCP-noMortar** | `use_mortar=False` | Mortar有無の効果定量化 |
| **NCP-noFriction** | `use_friction=False` | 摩擦なし基準線 |

### 適用ルール

- **新規ベンチマーク**: リファレンス構成で実施。他構成との比較が必要な場合はサブ構成も実行。
- **既存ベンチマーク**: S3パラメータチューニング等は、NCPソルバー版を追加する。AL版は参照用として残す。
- **リファレンス構成の更新**: ソルバーの改良に伴い、本ステータスの構成表を更新する。更新時は status ファイルで変更点を明記する。

## 2. S4 剛性比較ベンチマーク方針更新

### 現状の問題

status-097 の S4 は単体の梁要素 vs HEX8 の比較のみ:
- 矩形断面 HEX8 vs Timoshenko 梁（軸・曲げ・ねじり、線形小変形）
- 結論: 梁モデルが正確 → だが実用上の問題はここではない

### 更新方針: 撚線構造レベルの剛性比較を優先

実際のモデル選択に関わるのは「素線+被膜」「素線+シース」の**構造レベル**の等価剛性。
単体要素の精度は既に確認済みであり、次の段階として構造剛性の比較が必要。

### 優先比較対象

| 優先度 | モデル構成 | 比較内容 |
|--------|-----------|----------|
| **最高** | 素線+被膜（CoatingModel） | 被膜の剛性寄与、素線間接触を通じた等価剛性 |
| **最高** | 素線+シース（SheathModel） | シースの拘束効果、シース-素線間力伝達 |
| 高 | 素線+被膜+シース（フルモデル） | 全構成要素込みの等価剛性 |
| 参考 | 素線のみ（裸撚線） | 接触のみで保持される構造の剛性 |

### 対象モード（全構成で共通）

| モード | 荷重条件 | 抽出する等価剛性 |
|--------|----------|-----------------|
| **曲げ** | 片端固定・先端横荷重 | 等価曲げ剛性 EI_eq |
| **ねじり** | 片端固定・先端ねじりモーメント | 等価ねじり剛性 GJ_eq |
| **引張** | 両端引張 | 等価軸剛性 EA_eq |
| **圧縮** | 両端圧縮 | 等価圧縮剛性（座屈モード発生有無） |
| **大変形** | 上記モードを大変形領域（幾何学的非線形）で実施 | 荷重-変位曲線の非線形性 |

### 比較手法

各モデル構成 × 各モードで以下を比較:

1. **小変形域の等価線形剛性**: 初期接線剛性から抽出
2. **大変形の荷重-変位曲線**: 非線形領域での剛性低下・硬化
3. **モード間カップリング**: 曲げ-ねじり連成、引張による曲げ剛性変化
4. **文献値との比較**: Costello (1997), Foti & Martinelli (2016) 等の理論曲げ剛性

### 計測結果

#### 被膜の剛性寄与（0.1mm エナメル被膜, E=3GPa）

| モード | 裸撚線 k | 被膜付き k | 剛性比 | 断面剛性比 |
|--------|---------|----------|--------|----------|
| 引張 | 7.30e+06 N/m | 7.35e+06 N/m | 1.007 | EA: +0.32% |
| 曲げ | 3.80e+05 N/m | 3.83e+05 N/m | 1.007 | EI: +0.70% |
| ねじり | 41.4 N·m/rad | 41.6 N·m/rad | 1.007 | GJ: +0.67% |
| 圧縮 | — | 7.35e+06 N/m | — | — |

→ 被膜の剛性寄与は全モードで約0.3〜0.7%。接触力を通じた構造的寄与がやや上乗せ。

#### シースの剛性寄与（0.5mm PE/PVCシース, E=1.2GPa）

| パラメータ | 素線7本合計 | シース | 比率 |
|-----------|-----------|--------|------|
| EA | 4.40e+06 N | 1.23e+04 N | 0.28% |
| EIy | 1.10 N·m² | 0.065 N·m² | 5.92% |
| GJ | 0.85 N·m² | 0.046 N·m² | 5.50% |

→ シースは曲げ・ねじりで5〜6%の剛性寄与。軸方向は小さい。

## 確認事項

- NCPソルバーの19本以上での収束性は未検証。リファレンス構成として定義したが、大規模問題では追加のチューニングが必要になる可能性がある。
- S4剛性比較の大変形モードは接触NRの収束が前提。NCPソルバーでの大変形収束をまず確認する必要がある（CR梁アセンブラへの切替が必要）。
- 被膜モデルの接触半径拡大による貫入率増大（bare: 3%, coated: 12%）→ k_pen チューニングが要検討。
