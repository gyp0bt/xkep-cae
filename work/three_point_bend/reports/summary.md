- 形状: L=100mm, φ17mm, 20要素

## 所見

1. **status フィルタ必須**: なしでは frac=0.11 で停滞（Run 04, 05）
2. **k_pen 方式の影響は小**: 動的推定 vs 梁剛性で結果ほぼ同等（Run 02 vs 03）
3. **30mm 未到達**: 全 Run で frac≈0.89 が上限。dt_min 到達による打ち切り。
4. **荷重レベル**: E=25 → 132N（百数十N）、E=100 → 527N（数百N）。要求の「数百N」は E=100 で達成。
